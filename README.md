
# 🛡️ PhishGuard-AI  
AI-Powered Phishing URL Detection System  

PhishGuard-AI is a **cybersecurity project** that detects and analyzes phishing URLs using **machine learning** and a **deep scan engine**. It achieves **97.67% accuracy** on benchmark datasets and integrates **OpenPhish reputation checks** for advanced detection of sophisticated phishing attacks.  


## 🚀 Features
- 🔍 **Deep Scan Engine** – Crawls submitted URLs and analyzes whether they are legitimate or phishing.  
- 🤖 **AI Model** – Trained on UCI phishing dataset with >97% accuracy.  
- 🌐 **Reputation Check** – Integrates OpenPhish for domain reputation validation.  
- 📊 **Risk Scoring** – Generates a phishing probability score for each URL.  
- 🖥️ **Web Interface** – Simple UI for uploading and analyzing suspicious links.  
- ⚡ **Railway Deployment** – Ready for cloud deployment on [Railway.app](https://railway.app).

## 📦 Installation

bash

Run locally:
bash
